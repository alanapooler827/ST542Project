# ST 542 Project: CAFO Presence and Stream Water Quality

Student Consultants: Alana Pooler, Sarat Bantupalli, David Grant, Cameron Mullaney

## Project Overview

As student statistical consultants, we are working in collaboration with Dr. Ercumen to investigate associations between concentrated animal feeding operations (CAFOs) and stream water quality.

The analysis-ready data set includes multiple CAFO exposure metrics, such as CAFO count, spatial density, and distance from water quality monitoring stations. These metrics are measured at several spatial scales, including sub-watersheds, watersheds, and sub-basins. Because the data are nested across these geographic levels, the analysis will focus on parsimonious modeling approaches that appropriately account for clustering while helping identify which CAFO metrics are most informative.

The data set also includes several measures of stream water quality, including fecal coliform counts, nitrate/nitrite concentrations, and phosphorus concentrations. These measures will serve as the primary response variables in the analysis.

The data set contains monthly water quality measurements collected from 145 monitoring stations in the Cape Fear River Basin, with a total of 12,156 observations.
